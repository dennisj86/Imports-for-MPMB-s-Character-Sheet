# Legacy Character Panels

This folder contains legacy UI panels that are intentionally **not** part of the V2 default runtime path.

- `SpellSelectionPanel.tsx`
- `FeatSelectionPanel.tsx`

The default app flow (`/builder/:id`, `/sheet/:id`, `/content`) runs on the V2 architecture (`wizardV2`, `spellManagementService`, `characterEngine`, `mpmbCore`).
